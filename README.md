# lp-business-top
ビジネスサイトトップページ（サンプル）
[デモサイト](http://lp-example-2.masashisite.com/index.html)

### Install

Clone this repository.

```shell
$ git clone git@github.com:fktlnz/lp-business-top.git
```

Build static files.

```shell
$ cd lp-business-top

$ npm i

$ gulp
```

The browser will start automatically
